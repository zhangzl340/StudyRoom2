# 高校智能自习室系统开发计划

## 1. 项目概述

开发一个前后端分离的高校智能自习室系统，包括学生端和管理员端，实现自习室预约、签到签退、违规管理、数据统计等功能。

## 2. 技术栈

### 2.1 前端技术栈

* **核心框架**：Vue 3.3.4

* **状态管理**：Pinia 2.1.6

* **路由**：Vue Router 4.2.4

* **UI组件库**：Element Plus 2.3.3

* **网络请求**：Axios 1.4.0

* **图表库**：ECharts 5.6.0

* **日期处理**：Dayjs 1.11.9

* **WebSocket**：Socket.io-client 4.6.1

* **二维码**：qrcode.vue 3.4.1, vue-qrcode-reader 5.7.3

* **拖拽功能**：vue3-draggable-next 4.1.4

* **Excel处理**：xlsx 0.18.5

* **国际化**：vue-i18n 9.4.0

* **构建工具**：Vite 4.3.9

* **语言**：TypeScript 5.1.6

* **样式**：Sass 1.63.6

### 2.2 后端技术栈

* **后端框架**：Spring Boot 3.1.5

* **编程语言**：Java 17

* **数据访问**：MyBatis-Plus 3.5.4.1

* **安全框架**：Spring Security 6

* **认证方式**：JWT Token

* **缓存**：Redis

* **数据库**：MySQL 8.0

* **接口文档**：Swagger/Knife4j

* **工具库**：Lombok、Hutool、Apache Commons

* **开发工具：IntelliJ IDEA 2023**

## 3. 开发阶段

### 3.1 准备阶段

1. **环境搭建**

   * 安装Node.js、npm/yarn、Vue CLI

   * 安装JDK 17、Maven、MySQL 8.0

   * 配置开发环境变量

2. **项目初始化**

   * 前端：使用Vite创建Vue 3 + TypeScript项目

   * 后端：使用Spring Initializr创建Spring Boot项目

   * 配置项目依赖

3. **数据库设计**

   * 创建数据库和表结构

   * 配置数据库连接

### 3.2 后端开发

1. **基础框架搭建**
